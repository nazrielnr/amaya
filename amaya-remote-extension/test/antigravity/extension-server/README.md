# Extension Server Tests

This area is for WebSocket-based tests against the Amaya Remote extension server.

## Subfolders
- `models/`: model listing and model metadata checks.
- `ws-flow/`: state sync, send message, trace, and interactive flow tests.
- `mock-server/`: local mock server used to simulate extension events.

## Output Rule
- Keep WebSocket result JSON files inside the same extension-server subfolder that produced them.
