# Extension Models

Use this folder for extension-server model listing and model metadata tests.

## Typical Files
- `get-models.js`
- `test-all-models.js`
- `test-dump-models.js`
- `test-dump-raw-models.js`
- `test-model-configs.js`

## Output
- `out-models.json`, `discovered-models.txt`, and `user-status-response.json` stay here.
